# 📢Swear Word Detection

Speech to text app that beeps out the swearing.

### 📋Aims of App

- User can speak into the app and hear the voice clip
- The app translates the voice clip to text to detect swear wears
- App gives back a new voice clip with swear words beeped out

### 👩‍💻Technical Details

- Hugging Face
- React
- Next.js
